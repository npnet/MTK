111 /*
112  * USB sets an upper limit of 5 seconds for any transfer to be completed.
113  *
114  * Data originally from EHCI driver:
115  * Tested with some USB2.0 flash sticks:
116  * TUR turn around took about 2.2s for the slowest (13fe:3800), maximum
117  * of 250ms for the others.
118  *
119  * SET ADDRESS on xHCI controllers.
120  * The USB specification indicates that devices must complete processing
121  * of a SET ADDRESS request within 50 ms. However, some hubs were found
122  * to take more than 100 ms to complete a SET ADDRESS request on a
123  * downstream port.
124  */
125 #define USB_MAX_PROCESSING_TIME_US (1 * 1000 * 1000)
