uint32_t chip_select_hold_count

SPI master chip select hold count setting.

The unit is in SPI base clock. The range is from 0 to 32767. The chip select hold time is (cs_hold_count+1)*CLK_PERIOD.

◆ chip_select_idle_count

uint32_t chip_select_idle_count

SPI master chip select idle count setting.

The unit is in SPI base clock. The range is from 0 to 255. The chip select idle time between consecutive transaction is (cs_idle_count+1)*CLK_PERIOD.

◆ chip_select_setup_count

uint32_t chip_select_setup_count

SPI master chip select setup count setting.

The unit is in SPI base clock. The range is from 0 to 32767. The chip select setup time is (cs_setup_count+1)*CLK_PERIOD, where CLK_PERIOD is the cycle time of the clock the SPI engine adopts.
