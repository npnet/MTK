335 /********************* SPI Slave FSM *******************************
336 **cmd\status PWROFF PWRON ConfigRead(CR) ConfigWrite(CW)
337 ** Poweroff N Y Abnormal Abnormal
338 ** Poweron Y N N N
339 **ConfigRead N Y Abnormal Abnormal
340 ** Read N N Y Abnormal
341 **ConfigWrite N Y Abnormal Abnormal
342 ** Write N N Abnormal Y
343 
344 Note: N: invalid operation.
345  Y: the operation completed successfully.
346  Abnormal: an error occurred during the operation.
347 *******************************************************************/
