bool bt_mesh_flash_reset ( bt_mesh_flash_sector_t  sector)

This function is used to reset the mesh data stored in flash.

Parameters
[in]sectorindicates which flash sector of the mesh is to be reset.
Returns
true, means the mesh was successfully reset.
false, means the mesh reset failed.
Note
Please use bt_mesh_init instead of using this API individually.
