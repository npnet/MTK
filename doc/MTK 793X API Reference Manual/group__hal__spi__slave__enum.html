HAL_SPI_SLAVE_EVENT_RD_FINISH

HAL_SPI_SLAVE_EVENT_CWR_FINISH

HAL_SPI_SLAVE_EVENT_WR_FINISH

HAL_SPI_SLAVE_EVENT_WR_ERR

HAL_SPI_SLAVE_EVENT_TIMEOUT_ERR 

A timeout is detected between configure read command and read command or configure write command and write command.

HAL_SPI_SLAVE_EVENT_DMA_DONE 

All tx or rx data transfer done.
