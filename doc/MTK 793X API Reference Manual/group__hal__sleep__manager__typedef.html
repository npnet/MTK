typedef void(* hal_sleep_manager_callback_t) (void *data)

This typedef defines user's callback function prototype.

User should call hal_sleep_manager_register_suspend_callback() to register callback before entering sleep mode,
or call hal_sleep_manager_register_resume_callback() to register callback after leaving sleep mode.
Note, that the callback function is not appropriate for time-consuming operations.
parameter "data" : a user defined data used in the callback function.
