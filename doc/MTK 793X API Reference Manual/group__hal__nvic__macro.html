◆ HAL_NVIC_QUERY_EXCEPTION_NUMBER

#define HAL_NVIC_QUERY_EXCEPTION_NUMBER   ((SCB->ICSR & SCB_ICSR_VECTACTIVE_Msk) >> SCB_ICSR_VECTACTIVE_Pos)

This macro is used to query the current exception status of Cortex-M.

If the value is 0, the Cortex-M is running successfully, otherwise an exception occured and the exception number is the value of the macro.
