enum hal_sleep_mode_t

This enum defines the available sleep mode.

Enumerator
HAL_SLEEP_MODE_IDLE 

WFI: CPU idle only.

Wake up immediately w/o reboot to BROM

HAL_SLEEP_MODE_SLEEP 

Sleep: Deep Sleep, all mtcmos off, Xtal off, PMU set to 0.65v .

Wake up need to reboot to BROM

HAL_SLEEP_MODE_LEGACY_SLEEP 

Legacy Sleep: SYSRAM park to retention mode, power down unnecessary power for short wakeup response.
