Read data from eFuse logical addr.

Read a block size of data from logical address addr in eFuse into buffer buf.

Parameters
groupefuse group shoud be in range from 1 to 2. 1 for GRP2, 2 for GRP3.
addrlogical block address to read from.
bufefuse block size of data to read to.
Returns
HAL_EFUSE_INVALID_ACCESS the efuse block address doesn't exist in the mapping table(un-blown).
HAL_EFUSE_INVALID_PARAMETER group is not supported.

◆ hal_efuse_physical_write()

hal_efuse_status_t hal_efuse_physical_write ( uint32_t  group,
uint32_t  addr,
uint32_t *  buf 
)

Write data into eFuse.

This API just provide for GRP1.

Write a block size of data in buffer buf into eFuse at the physical address addr.

Parameters
groupefuse group should be 0 for GRP1.
addreeprom block address to write. example : eeprom address 0x0b0 under GRP1 is equal to 0x0b0 logical address of GRP1.
bufefuse block size of data to write from.
Returns
HAL_EFUSE_OK if write succeeded.
HAL_EFUSE_INVALID_PARAMETER group is not supported.
HAL_EFUSE_INVALID_ACCESS efuse already be blown, address is not supported.
