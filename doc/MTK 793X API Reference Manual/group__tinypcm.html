Overview

ALSA uses the ring buffer to store outgoing (playback) and incoming (capture, record) samples.

There are two pointers being maintained to allow a precise communication between application and device pointing to current processed sample by hardware and last processed sample by application. The modern audio chips allow to program the transfer time periods. It means that the stream of samples is divided to small chunk. Device acknowledges to application when the transfer of a chunk is complete.
