Hardware and software environment
  • Supported platform
    • MediaTek MT7933 HDK.
  • HDK switches and pin configuration
    • The FTDI debug board can transfer USB interface to UART interface. Using this debug board can debug through UART, transmit and receive a signal form PC.
    • Two AMIC
    • J10 and J11 provide the pin for Speaker
    • Wi-Fi and BT function MT7933 main board reserve a Wi-Fi SMA connector and a BT SMA connector. Please connect external antenna to transmit and receive RF signals
    • There are some buttons on the board:
      • SW1 - reset.
      • SW2 - Vol+
      • SW3 - Vol-
      • SW4 - RTC_EINT
      • SW6 - DIP Switch : Switch board to Normal mode or Download mode.
    • USB slot
      • CON8 - CON8 is a USB 5V power for MT7933 main board, or you can use external 5V power at J1.
      • CON6 - CON6 is a USB function com port which do not supply 5V power to main board.
  • Environment configuration
    • A serial tool is required, such as hyper terminal or TeraTerm for UART logging.
    • COM port settings. baudrate: 921600, data bits: 8, stop bit: 1, parity: none and flow control: off.
  • See more information in MT7933_RFB_Users_Guide.pdf to download code.

Run the examples
  • Connect the FTDI board to a PC with a USB cable.
  • Build the example project and download the binary file to the MT793x HDK. See more information in MT793x_Flash_Burning_Tool_v2.77_ext.docx to Download code.
  • Reboot the HDK, the console will show "$" message to indicate the HDK is booting up.
  • Use '?' and enter to query the available command line options. Note that the command line options are still under development and subject to change without notice.
  • Below example demonstrate the how to mount USB and play a mp3 file.
  • Example 1. MP3 playback demo
    • Prepare an U-disk, put in the mp3 file which you want to play.
    • Prepare 2 Micro USB cables, 1 OTG cable and the U-disk which contains mp3 files, connect to RFB.
    • After power on, please enable usb first.
      ssusb enable
    • To make sure file system on USB works well, please type "ff mount USB" and "ff ls USB:/" in order. If file system works well, it will list all files stored in the U-disk.
      ff mount USB
      ff ls USB:/
    • Now we can play mp3 files, please use "mp3 play USB:/[filename]" to play audio. For example, if I want to play Stairway_To_Heaven.mp3, I would type "mp3 play USB:/Stairway_To_Heaven.mp3".
      mp3 play USB:/Stairway_To_Heaven.mp3
      audio ( "mp3 pause" ) and resume the audio ( "mp3 resume" ).
      mp3 pause
      mp3 resume
