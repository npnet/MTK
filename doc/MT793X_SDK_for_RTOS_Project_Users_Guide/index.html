Introduction
Overview
  • This file guides you through the common and specific folder and file structure of available example projects. The example projects and applications are under [sdk_root]/project folder of the current SDK.
Common folders and files
  • The common folders and files in example projects are listed below.
    • readme.txt: The first document to guide you through using this project, including setup, usage and structure of this project.
    • GCC: Build related files, including the compiler dependent files and project makefiles
      • feature.mk: The configuration files of feature
      • hal_feature.mk: The configuration files of peripheral driver
    • inc: The header files of this project including the following common header files.
      • ept*: The configuration files generated by Easy Pinmux Tool (EPT), with the *.ews file under ept_ews folder of a project
      • FreeRTOSConfig.h: FreeRTOS configuration. (Not in hal_example projects)
      • sys_init.h: The header file of sys_init.c initializing the system. (Not in hal_example projects)
    • src: The source files of this project.
Directory contents
  • Example projects using MT793x HDK
    • mt793x_hdk/apps: Applications with FreeRTOS on MT793x.
    • mt793x_hdk/apps/qfn_sdk_demo: A reference application for MT7931 board with CLI to demonstrate the basic functionalities of WIFI, Bluetooth, and so forth.
    • mt793x_hdk/apps/bga_sdk_demo: A reference application for MT7933 board with CLI to demonstrate the basic functionalities of WIFI, Bluetooth, Audio, and so fort
    • mt793x_hdk/apps/qfn_tfm_demo: A reference application for MT7931 board with CLI to demonstrate the basic functionalities of Security feature.
    • mt793x_hdk/apps/bga_tfm_demo: A reference application for MT7933 board with CLI to demonstrate the basic functionalities of Security feature.
    • mt793x_hdk/apps/audio_ref_design: A reference application for MT7933 board with CLI to demonstrate the basic functionalities of MP3 Player.
