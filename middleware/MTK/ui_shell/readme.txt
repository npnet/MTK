UI shell module usage guide

Brief:          This module is the UI shell.
Usage:          GCC:  Include the module with "include $(SOURCE_DIR)/middleware/MTK/ui_shell/module.mk" in your GCC project Makefile.
                KEIL: Drag the middleware/MTK/ui_shell folder to your project.
                      Add middleware/MTK/ui_shell/inc to include paths.
                IAR:  Drag the middleware/MTK/ui_shell folder to your project. Add middleware/MTK/usb/inc to include paths.
                      Please add middleware/MTK/ui_shell/inc to include paths if USB support is needed.
Dependency:     none.
Notice:         none.
Relative doc:   none.
